# Java Swing Calculator

A simple desktop calculator application built using Java Swing. The calculator provides basic arithmetic operations with an easy-to-use graphical user interface.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Decimal number support
* Clear button to reset calculations
* User-friendly GUI built with Java Swing
* Automatic removal of unnecessary decimal places (e.g., displays `10` instead of `10.0`)

## Technologies Used

* Java
* Java Swing
* AWT Event Handling

## Project Structure

```
Calculator.java
```

The entire calculator functionality, GUI design, and event handling are implemented in a single Java file.

## How to Run

### Prerequisites

* JDK 8 or later installed
* Any Java IDE (Eclipse, IntelliJ IDEA, VS Code) or Command Prompt

### Compile

```bash
javac Calculator.java
```

### Run

```bash
java Calculator
```
